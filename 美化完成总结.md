## ❓ 常见问题

### Q1: 编译时提示找不到头文件
**A**: 检查 Qt 环境变量是否正确配置，确保 PATH 包含 Qt 的 bin 目录

### Q2: 编译成功但运行崩溃
**A**: 检查是否以管理员权限运行，查看日志文件中的错误信息

### Q3: 点击新主题按钮无反应
**A**: 
1. 确认按钮的 objectName 正确（区分大小写）
2. 检查信号和槽是否正确连接
3. 查看应用程序输出面板的错误信息

### Q4: 主题切换后界面无变化
**A**: 
1. 检查 ThemeManager 是否正确初始化
2. 查看日志输出
3. 尝试重启应用程序

## 💡 提示

### 使用 Qt Creator 的技巧

1. **自动保存**: 启用自动保存功能，避免丢失修改
2. **实时预览**: 按 Ctrl+P 在设计器中预览 UI
3. **快速导航**: 使用 F4 在头文件和实现文件间切换
4. **代码补全**: Qt Creator 会自动提供代码补全
5. **查找引用**: 选中函数名后按 Ctrl+Shift+U 查找引用

### 编译优化

1. **并行编译**: Qt Creator 默认使用多线程编译
2. **增量编译**: 只重新编译修改过的文件
3. **预编译头**: Qt 自动使用预编译头加速编译
